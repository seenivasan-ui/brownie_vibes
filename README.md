## How to Add Images

1. Create an `images/` folder inside the `brownie-vibes/` folder
2. Add your photos using the exact filenames listed above
3. Open `index.html` in your browser — images will show automatically

**Accepted formats:** .jpg, .jpeg, .png, .webp
**Recommended size:** 600×400px or larger (landscape works best)

## How to Change Prices / Descriptions

1. Open `script.js` in VS Code
2. Find the `menuData` array at the top
3. Edit the `price` or `desc` for any item
4. Save the file — changes show on the website immediately
